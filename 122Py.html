1.2.2 Catch-A-Turtle

In the 2022-2023 school year during the AP CSP course, I completed the 'Catch A Turtle' project. This project featured a dynamic game where clicking the start button initiated a timer and score display, along with a circular turtle element.

Players scored points by clicking the turtle as it moved to different locations. High-scoring players earned a spot on the displayed leaderboard when the timer expired.

Working on this project improved my skills in managing multiple Python files, reinforced my understanding of pair programming, and deepened my knowledge of the Turtle graphics module. This project was a collaborative effort with Matthew Loo.
